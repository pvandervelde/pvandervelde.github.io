Ops-Tools-Build release - V0.3.0

Sunday, July 16, 2017 | Posted in
Ops-Resource Ops-Tools-Build

Version 0.3.0 of the Ops-Tools-Build toolkit was released.

This release was focussed on providing the capabilities to use the Docker toolset to create new containers and to run validation of VM resources generated by Packer. All the work items that have been closed for this release can be found on GitHub.

New functionality

  • 33 - Added the ability to create ISO files during the test stage.
  • 24 - Added functions to invoke docker to create container resources.
  • 32 - Added functions to validate a Packer image by executing packer with the previously created image and executing tests during the packer test build.
